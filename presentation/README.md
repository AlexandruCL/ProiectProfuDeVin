# Presentation

This folder contains a presentation outline that you can paste into Canva or generate as a `.pptx`.

## Files
- [deck.md](deck.md) — slide-by-slide content (Canva-friendly)
- [generate_pptx.py](../scripts/generate_pptx.py) — optional script to generate a real PowerPoint `.pptx`

## Generate PowerPoint (.pptx)
From repo root:
1. Install:
   - `pip install python-pptx`
2. Run:
   - `python scripts/generate_pptx.py`
3. Output:
   - `presentation/ProiectProfuDeVin_Deployment.pptx`

## Canva
Canva can import `.pptx`. Upload the generated file into Canva and adjust theme/fonts.
